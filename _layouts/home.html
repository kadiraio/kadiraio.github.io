Benefits of using Kadira

Make your apps faster

Our recommendation system will tell you which methods and publications you should focus on to improve response times.

Then you can use traces to identify the bottlenecks and you can apply fixes using our guides

Send less data over the network

Sending more data with your publications makes your apps slower, especially for mobile networks. You can track this latency with "Network Latency".

Our recommendation system will tell you which publications you should focus on. Use our guides to reduce the amount of data you send to improve "Network Latency".

Reduce CPU and Network Usage

If you have many subscriptions, you will have many observers and they will affect the performance of the CPU and the network.

However, Meteor can reuse observers for identical queries. You can track the reuse of observers directly with our dashboard.

Sort publications by "Low Observer Reuse" & improve reuse with our guides.

Better Utilization of Server Memory

Meteor maintains a copy in memory of all your data sent to each client. This is normally not an issue, but there are certain situations you need to consider.

We show you the estimated memory usage for each publication and how much of a document has been updated.

Using these metrics and our guides, you can optimize the memory usage of your application.

Quickly Debug and Fix Errors

We have taken tracking errors to the next level. Now you can debug an error with its context and identify the root cause quickly.

Improve User Experience

Normally, Iron Router unsubscribes all the current route's subscriptions before moving to another route. For each route change, users need to wait until the data has been returned from the server, which adds an unnecessary server load.

You can track those publications by sorting "Shortest Lifespan" and use our guides on improving this behaviour.

Real-time Performance Overview

Our real-time dashboard shows the overall performance of an app in a single view. This is perfect for a wall-mounted display.
